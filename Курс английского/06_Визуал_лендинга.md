## 6. АНИМАЦИЯ

### Типы анимаций:

#### 1. Появление при скроллинге (Fade In Up):
```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(40px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
- **Применение:** Все секции, карточки, текстовые блоки
- **Длительность:** 0.6s
- **Задержка:** 0.1s между элементами
- **Easing:** `cubic-bezier(0.25, 0.46, 0.45, 0.94)`

#### 2. Увеличение при наведении (Scale):
```css
.card:hover {
  transform: scale(1.02);
  box-shadow: 0 8px 32px rgba(0,0,0,0.12);
}
```
- **Применение:** Карточки преимуществ, тарифов
- **Длительность:** 0.3s
- **Easing:** `ease-out`

#### 3. Пульсация кнопки (Pulse):
```css
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.05);
  }
}
```
- **Применение:** CTA-кнопки (при загрузке страницы)
- **Длительность:** 2s
- **Повтор:** 3 раза

#### 4. Счётчики (Count Up):
- **Применение:** Числа в блоке социального доказательства
- **Длительность:** 2s
- **Easing:** `ease-out`

#### 5. Таймер (Countdown):
- **Применение:** Обратный отсчёт
- **Длительность:** Постоянно (каждую секунду)
- **Эффект:** Мерцание при смене цифр

#### 6. Плавная прокрутка (Smooth Scroll):
```css
html {
  scroll-behavior: smooth;
}
```
- **Применение:** Все внутренние ссылки

### Правила анимаций:
- **Не перегружать** — максимум 2-3 анимации на экран
- **Не блокировать скролл** — анимации только визуальные
- **Уважать `prefers-reduced-motion`** — отключать для пользователей с ограниченными возможностями
- **Оптимизация** — использовать `transform` и `opacity` (не `top/left`)

## 11. ФИНАЛЬНЫЕ РЕКОМЕНДАЦИИ

### Для премиального вида:
1. **Много воздуха** — не бойся пустого пространства
2. **Минимум цвета** — чёрный + белый + 1 акцент
3. **Чистые линии** — никаких декоративных элементов
4. **Единый стиль** — все элементы одного «семейства»
5. **Качественные фото** — без стоковых «улыбающихся бизнесменов»

### Для конверсии:
1. **Контраст CTA** — кнопка должна быть самой яркой
2. **Повтор CTA** — в Hero, после проблем, после кейсов, в конце
3. **Движение** — анимации привлекают внимание
4. **Доверие** — фото реальных людей, а не стоковые

### Чеклист перед запуском:
- [ ] Все кнопки кликаются на мобильном
- [ ] Форма отправляется и валидируется
- [ ] Таймер работает корректно
- [ ] Фото оптимизированы (WebP, < 200KB)
- [ ] Скорость загрузки < 3 секунд
- [ ] A/B тест Hero-заголовка настроен
