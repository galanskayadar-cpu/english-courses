## Варианты деплоя

### 1. GitHub Pages (бесплатно)
```bash
# Инициализация git
git init
git add .
git commit -m "Initial commit"

# Создание репозитория на GitHub
# Затем:
git remote add origin https://github.com/username/english-courses.git
git push -u origin master

# В настройках репозитория: Settings → Pages → Source: master branch
```

### 2. Netlify (бесплатно)
- Загрузить папку на netlify.com
- Автоматический деплой

### 3. Vercel (бесплатно)
```bash
npm i -g vercel
vercel
```

### 4. Любой хостинг
- Загрузить файлы через FTP/FileZilla
- index.html должен быть в корне
